A simple and efficient web server built with Go, serving static files and handling form submissions.

## Features

- Serves static files from the `./static` directory
- Handles `/hello` route for greetings
- Handles `/form` route for form submissions

## Installation

1. Clone the repository:
```bash
git clone https://github.com/zineeddinehazi/go-server.git
```
2. Navigate to the project directory:
```bash
cd go-server
```
3. Run the server:
```bash
go run main.go
```

## Usage

- Access the server at `http://localhost:8080`
- Visit `/hello` to see a greeting
- Visit `/form.html` to submit a form
